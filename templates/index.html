Model Architecture

The system utilizes MobileNetV2, a highly efficient convolutional neural network optimized for mobile and edge devices. It relies on inverted residual blocks and depthwise separable convolutions to achieve high accuracy with minimal computational cost.

  • Input Shape: 224x224x3
  • Feature Extraction: MobileNetV2 Base
  • Classification Head: Global Average Pooling + Dense Layers
  • Optimization: Transfer Learning
